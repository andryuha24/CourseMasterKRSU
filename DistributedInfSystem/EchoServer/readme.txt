Задание: 1/ Мulti-client echo server
Алгоритм: Запустить Сервер, Запустить Клиент, Отправлять сообщения.
	У Клиента и Сервера автоматически создается лог файл, куда записываются
	все входящие и исходящие сообщения.
	Клиент реализован без ввода имени.
	Любой клиент может подключаться и отключаться в любой момент времени, пока работает сервер.
	Реализован "медленный" клиент.
Пункты меню: На клиенте - Изменить адрес сервера - Выбор клавишей F1
			  Медленный/Обычный клиент - - Выбор клавишей F2
			  Добавить текстовый файл с сообщениями, для отправки всем остальным - Выбор клавишей F3
	     На сервере - Изменить адрес сервера - Выбор клавишей F1
